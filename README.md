# オデュッセイア 縦書きビュワー

ホメロス『オデュッセイア』(Samuel Butler の散文英訳 / Project Gutenberg #1727 を底本とした
現代日本語訳)を、**読みやすい縦書き(tategaki)** で読むための Web ビュワーです。
`Odyssey/02_和訳`(全24歌の和訳)と `Odyssey/04_事典`(人物・地名の索引)の Markdown を
そのままソースとして表示します。

## ▶ ビュワーを開く

**<https://tomosud.github.io/Odyssey/>**

- 和訳は縦書き(右→左)で表示。ホイール/矢印キー(←→)・下部のリンクで歌を移動できます。
- 事典は横書きの一覧で、名前・原文表記・説明をまたいで**絞り込み検索**できます。
- ライト/ダークテーマ、PC・スマートフォンに対応。

## 仕組み

- `index.html` … ビュワーのエントリ(サイトのトップ)
- `viewer/viewer.css`, `viewer/viewer.js` … 表示ロジック(外部ライブラリ非依存)
- `viewer/manifest.json` … 目次データ(各歌のタイトルと Markdown へのパス)
- `.nojekyll` … GitHub Pages に Markdown を加工せずそのまま配信させるためのファイル

`viewer/manifest.json` は各和訳ファイルの frontmatter(`heading_ja` 等)から生成しています。
歌を追加・改題したときは manifest を更新してください。

> 翻訳・整備プロジェクトそのものの記述ルールは [`Odyssey/README.md`](Odyssey/README.md) を参照。
