# 人物索引

> [!info]
> 和訳で登場した人物・神を、登場のつど追加していく(五十音順)。「原文表記」はButler英訳の表記(ローマ神名)。「登場」は登場・言及のあった歌。

| 名前 | 原文表記 | 説明 | 登場 |
|---|---|---|---|
| アイギストス | Aegisthus | アガメムノンの留守中にその妻と通じ、帰国した彼を殺した男。のちにオレステスに討たれる。「身から出た錆」の実例として神々が引き合いに出す | 1 |
| アイギュプティオス | Aegyptius | イタケの長老。息子アンティポスをキュクロプスに殺され、今も嘆いている。20年ぶりの集会で最初に発言 | 2 |
| アガメムノン | Agamemnon | トロイア遠征のギリシャ軍総大将。帰国した日にアイギストスに殺された | 1 |
| アテナ | Minerva | 知恵と戦の女神。ゼウスの娘。オデュッセウス父子の守護者。メンテス(1歌)、メントル(2歌)、テレマコス本人(2歌)に変身して行動する | 1, 2 |
| アトラス | Atlas | 天と地を分かつ大柱を支える神。カリュプソの父 | 1 |
| アルクメネ | Alcmena | 昔の名高い女性(ヘラクレスの母)。アンティノオスがペネロペと比較して言及 | 2 |
| アンキアロス | Anchialus | メンテスの父 | 1 |
| アンティノオス | Antinous | 求婚者の筆頭格。エウペイテスの子。攻撃的で、テレマコスを正面から侮辱する | 1, 2 |
| アンティポス | Antiphus | アイギュプティオスの息子。オデュッセウスとともに出征し、ポリュペモスの洞窟で最後の犠牲になった | 2 |
| イカリオス | Icarius | ペネロペの父。「母を実家に帰せ」という議論で繰り返し言及される | 1, 2 |
| イロス(メルメロスの子) | Ilus | エピュラの人。神々を恐れて、オデュッセウスに矢毒を渡さなかった。※後の歌に出る乞食イロスとは別人 | 1 |
| エウエノル | Evenor | レイオクリトスの父 | 2 |
| エウペイテス | Eupeithes | アンティノオスの父 | 1, 2 |
| エウリュクレイア | Euryclea | 忠実な老家政婦・乳母。オプスの娘。若いころラエルテスが牛20頭の値で買った。テレマコスを赤ん坊から育て、旅立ちの秘密を守る | 1, 2 |
| エウリュノモス | Eurynomus | アイギュプティオスの息子。求婚者の一人 | 2 |
| エウリュマコス | Eurymachus | 求婚者の有力者。ポリュボスの子。弁が立ち、老預言者を恫喝する | 1, 2 |
| エリニュエス | Erinyes | 復讐の女神たち。特に親族に対する罪を追及する。母を追い出せばその呪いを買う、とテレマコスが恐れる | 2 |
| オデュッセウス | Ulysses | 主人公。イタケの王。トロイア戦争の英雄で、知略で知られる。出征から20年、いまだ帰郷できずにいる | 全編 |
| オプス | Ops | エウリュクレイアの父。ペイセノルの子 | 1, 2 |
| オレステス | Orestes | アガメムノンの息子。父の仇アイギストスを討って名を上げた。テレマコスの模範として繰り返し言及される | 1 |
| カリュプソ | Calypso | オギュギエ島に住む女神。アトラスの娘。オデュッセウスを洞窟に住まわせ、夫にしようと引き留めている | 1 |
| クロノス | Saturn | ゼウスの父。「クロノスの子」はゼウスの呼び名 | 1 |
| ゼウス | Jove | 最高神。「神々と人間の父」 | 1, 2 |
| テミス | Themis | 掟の女神。「集会の始めと終わりを司る」 | 2 |
| テレマコス | Telemachus | オデュッセウスとペネロペの一人息子。父を知らずに育ち、物語の冒頭で大人への一歩を踏み出す | 1, 2 |
| テュロ | Tyro | 昔の名高い女性。アンティノオスがペネロペと比較して言及 | 2 |
| トオサ | Thoosa | ニンフ。ポルキュスの娘。ポセイドンとの間にポリュペモスを産んだ | 1 |
| ネストル | Nestor | ピュロスの老王。トロイア帰りの生き証人として、テレマコスが訪ねるべき相手 | 1, 2(言及) |
| ノエモン | Noemon | プロニオスの子。頼まれてテレマコスに船を貸した | 2 |
| ハリテルセス | Halitherses | イタケ一の老預言者・鳥占い。出征時に「20年目に誰にも気づかれず帰る」と予言し、集会でそれを再言する | 2 |
| ヒュペリオン | Hyperion | 太陽神。オデュッセウスの部下たちはその牛を食べたために滅んだ | 1 |
| ピュロスの伝令ペイセノル | Pisenor | 集会でテレマコスに発言者の杖を渡したイタケの伝令。※エウリュクレイアの祖父ペイセノルとは別人 | 2 |
| ペイセノル(オプスの父) | Pisenor | エウリュクレイアの祖父 | 1, 2 |
| ペミオス | Phemius | イタケの吟遊詩人。求婚者たちに強いられて歌う | 1 |
| ペネロペ | Penelope | オデュッセウスの妻。イカリオスの娘。「昼に織り、夜にほどく」織物の計略で求婚者を3年あざむいた | 1, 2 |
| ヘルメス | Mercury | 神々の使者。アイギストスへの警告役として言及 | 1 |
| ポセイドン | Neptune | 海の神。ポリュペモスの父。息子の目をつぶされた恨みで、オデュッセウスの帰郷を妨げ続ける | 1 |
| ポリュペモス | Polyphemus | 一つ目の巨人キュクロプスの王。ポセイドンとトオサの子。オデュッセウスに目をつぶされた | 1, 2(言及) |
| ポリュボス | Polybus | エウリュマコスの父 | 1, 2 |
| ポルキュス | Phorcys | 海の王(海神)。トオサの父 | 1 |
| ミュケネ | Mycene | 昔の名高い女性。アンティノオスがペネロペと比較して言及 | 2 |
| ムーサ | Muse | 詩と歌の女神。叙事詩の冒頭で語り手が呼びかける | 1 |
| メネラオス | Menelaus | スパルタ王。アガメムノンの弟。アカイア勢で最後に帰国した男として、テレマコスが訪ねるべき相手 | 1, 2(言及) |
| メンテス | Mentes | タポス人の王。アンキアロスの子。第1歌でアテナが借りた姿 | 1 |
| メントル | Mentor | オデュッセウスの旧友。出征時に家のいっさいを託された。第2歌からアテナがこの姿を借りる(英語 mentor の語源) | 2 |
| ラエルテス | Laertes | オデュッセウスの父。息子の出征後は町を離れ、田舎のぶどう畑で貧しく暮らす | 1, 2(言及) |
| レイオクリトス | Leiocritus | 求婚者。エウエノルの子。メントルを嘲って集会を解散させた | 2 |
